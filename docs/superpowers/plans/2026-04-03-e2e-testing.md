# E2E Testing Setup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a `packages/e2e` workspace package with a tiny demo app and Playwright tests wired to `@playwright-cart/reporter`, enabling full local E2E verification without publishing to npm.

**Architecture:** A new private pnpm workspace package (`@playwright-cart/e2e`) that depends on `@playwright-cart/reporter: workspace:*`. Playwright's `webServer` serves a static Todo demo app; a `pretest` script builds the reporter before each test run. Tests run against the local docker-compose stack (server on `:3001`, web on `:80`).

**Tech Stack:** Playwright Test, @playwright-cart/reporter (workspace), serve (static file server), Docker Compose

## Task 2: Create the demo app

**Files:**
- Create: `packages/e2e/demo-app/index.html`

- [ ] **Step 1: Create the static Todo app**

Use `createElement` and `textContent` throughout — never `innerHTML` with user-provided text.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Demo Todo App</title>
  <style>
    body { font-family: sans-serif; max-width: 500px; margin: 40px auto; padding: 0 20px; }
    h1 { font-size: 1.5rem; }
    .input-row { display: flex; gap: 8px; margin-bottom: 16px; }
    input { flex: 1; padding: 8px; border: 1px solid #ccc; border-radius: 4px; font-size: 1rem; }
    button { padding: 8px 12px; cursor: pointer; border-radius: 4px; border: none; font-size: 0.9rem; }
    .add-btn { background: #2563eb; color: white; }
    ul { list-style: none; padding: 0; margin: 0; }
    li { display: flex; align-items: center; gap: 8px; padding: 10px 0; border-bottom: 1px solid #eee; }
    li.done > span { text-decoration: line-through; color: #999; }
    li > span { flex: 1; }
    .done-btn { background: #16a34a; color: white; }
    .del-btn { background: #dc2626; color: white; }
  </style>
</head>
<body>
  <h1>Todo</h1>
  <div class="input-row">
    <input data-testid="todo-input" type="text" placeholder="Add a new todo..." />
    <button data-testid="add-button" class="add-btn">Add</button>
  </div>
  <ul data-testid="todo-list"></ul>

  <script>
    const input = document.querySelector('[data-testid="todo-input"]');
    const addBtn = document.querySelector('[data-testid="add-button"]');
    const list = document.querySelector('[data-testid="todo-list"]');

    function addTodo() {
      const text = input.value.trim();
      if (!text) return;

      const li = document.createElement('li');
      li.dataset.testid = 'todo-item';

      const span = document.createElement('span');
      span.textContent = text;

      const doneBtn = document.createElement('button');
      doneBtn.className = 'done-btn';
      doneBtn.dataset.testid = 'complete-button';
      doneBtn.textContent = 'Done';

      const delBtn = document.createElement('button');
      delBtn.className = 'del-btn';
      delBtn.dataset.testid = 'delete-button';
      delBtn.textContent = 'Delete';

      doneBtn.addEventListener('click', () => { li.classList.toggle('done'); });
      delBtn.addEventListener('click', () => { li.remove(); });

      li.appendChild(span);
      li.appendChild(doneBtn);
      li.appendChild(delBtn);
      list.appendChild(li);
      input.value = '';
    }

    addBtn.addEventListener('click', addTodo);
    input.addEventListener('keydown', (e) => { if (e.key === 'Enter') addTodo(); });
  </script>
</body>
</html>
```

## Task 3: Create Playwright config

**Files:**
- Create: `packages/e2e/playwright.config.ts`

> **Important:** The reporter source detects the HTML reporter by checking `r[0] === 'html'` on `config.reporter` in `onBegin` (`packages/reporter/src/index.ts:56`). Use the string `'html'` for the built-in reporter so this detection works and the HTML report zip is uploaded. Use the imported class for `PlaywrightCartReporter`.

- [ ] **Step 1: Create playwright.config.ts**

```typescript
import { defineConfig } from '@playwright/test'
import PlaywrightCartReporter from '@playwright-cart/reporter'

export default defineConfig({
  testDir: './tests',
  reporter: [
    ['html', { open: 'never' }],
    [PlaywrightCartReporter, {
      serverUrl: 'http://localhost:3001',
      project: 'e2e-demo',
      branch: process.env.BRANCH ?? 'local',
      commitSha: process.env.COMMIT_SHA ?? 'manual',
    }],
  ],
  use: {
    baseURL: 'http://localhost:5500',
    trace: 'on',
  },
  webServer: {
    command: 'npx serve demo-app -l 5500',
    url: 'http://localhost:5500',
    reuseExistingServer: true,
  },
})
```

## Task 4: Create passing tests

**Files:**
- Create: `packages/e2e/tests/todo.spec.ts`

- [ ] **Step 1: Create todo.spec.ts**

```typescript
import { test, expect } from '@playwright/test'

test('homepage loads with correct title', async ({ page }) => {
  await page.goto('/')
  await expect(page).toHaveTitle('Demo Todo App')
})

test('can add a todo item', async ({ page }) => {
  await page.goto('/')
  await page.fill('[data-testid="todo-input"]', 'Buy milk')
  await page.click('[data-testid="add-button"]')
  await expect(page.locator('[data-testid="todo-list"]')).toContainText('Buy milk')
})

test('can mark a todo item as complete', async ({ page }) => {
  await page.goto('/')
  await page.fill('[data-testid="todo-input"]', 'Walk the dog')
  await page.click('[data-testid="add-button"]')
  await page.click('[data-testid="complete-button"]')
  await expect(page.locator('[data-testid="todo-item"]')).toHaveClass(/done/)
})

test('can delete a todo item', async ({ page }) => {
  await page.goto('/')
  await page.fill('[data-testid="todo-input"]', 'Take out trash')
  await page.click('[data-testid="add-button"]')
  await page.click('[data-testid="delete-button"]')
  await expect(page.locator('[data-testid="todo-list"]')).not.toContainText('Take out trash')
})
```
